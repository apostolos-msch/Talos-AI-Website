# Talos Startup - AI-Powered Out-of-Court Mechanism Automation

**Talos Startup** is a fintech web platform designed to streamline and automate legal-financial out-of-court mechanisms. The project was developed as an academic assignment for the **University of Macedonia (UoM)**, specifically for the "Mobile Business and E-Commerce Technologies" lab.

## üöÄ Project Overview

The project demonstrates a multi-layered web architecture, including standard desktop views, responsive design, and a dedicated mobile-first experience using jQuery Mobile.

### Key Features

* **Agentic AI Integration**: Utilizes hybrid architecture with GPT-4o for data extraction and deterministic Python algorithms for financial accuracy.
* **Legal Compliance**: Built-in RAG (Retrieval-Augmented Generation) technology connected to official Greek legal databases (FEK 4738/2020).
* **Security**: Developed with high-security standards suitable for Azure environments.
* **Human-in-the-Loop**: Systems designed to allow expert oversight for final proposal approvals.
